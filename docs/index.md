### Project Summary
The goal of this project is to develop a fully functional subterranean rover that can drill into the ground, detect underground magnetic variations, and collect targeted soil samples from a variety of environments. The rover uses a motor driven drilling system to break through different soil layers and gather material for analysis. A Hall effect sensor is included to measure changes in magnetic fields, allowing the rover to identify metallic objects or magnetic anomalies beneath the surface. Wireless communication provides remote control and data exchange, while the human machine interface gives the operator clear feedback and the ability to guide the rover based on real time information. Together, these subsystems create a reliable and efficient platform for exploring underground environments and collecting meaningful data.

### My Contribution

My primary responsibility is the development and integration of the Hall effect sensor subsystem. This subsystem allows the rover to detect underground magnetic variations, identify metallic objects, and provide the operator with meaningful data that influences drilling and sampling decisions.

My contribution includes the electrical integration of the Hall effect sensor, the firmware required to read and process magnetic field measurements, and the communication logic that sends this data to the rest of the rover through the team’s wireless messaging protocol. This work directly supports the rover’s ability to navigate subsurface environments, determine sampling locations, and operate safely and efficiently. Readers interested in sensing, data acquisition, or subsystem communication will find the details of my module particularly relevant.
